Features:
==========
a) Five star rating system that can be added in to any web page to rate articles, tutorials, photos and images.
b) Database based script written in PHP and Jquery to rate your web page.
c) The 5 star script checks for duplicate votes by checking the clients IP address.
d) The jquery based script rating is easy to deploy to the websites.
e) Easy to install and configure.
Say you have added the five star rating system in two pages page1 and page2. and the user rates the page1 as 4, the rating will be considered for page1 only.The rating for page2 would be zero until you rate it.
Note: Your *.html file should be changed as *.php files. 


Installation:
==============

a) unzip five-star-rating.zip.

File permissions:
a) You will get respective files and "images" folder.
b) Set read, write permission for the file config.php for Linux
users.

Creating a database:
===================
a) Create a database for Five star rating Script, namely "star".
b) Change the configuration file(config.php) to match with your database.
c) Run the file "tables.php" in your browser.
d) You will get the message as "Your tables are created succezzfully.........."

Embedding the Five star rating script:
a) Run the file "code.php" in your browser
b) Copy the code and insert into the page where ever you want this "JQuery Five Star
Rating Systems" to be displayed.

Important Note:
==============
NOTE 1: Your file's should be named with .php extension. [if your filename is test.html rename it to test.php]
